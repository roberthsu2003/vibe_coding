## 📖 什麼是 Vibe Coding？

「Vibe coding」是一種革命性的程式設計方式，透過大型語言模型（LLM）等 AI 工具，讓開發者能以自然語言描述需求來生成程式碼。這種方法讓程式設計變得更加直觀和高效，無論是專業開發者還是程式新手都能快速上手。

### 🎯 核心特色
- **自然語言輸入**：用日常語言描述需求，AI 即時生成對應程式碼
- **降低門檻**：讓非專業程式設計者也能快速開發軟體
- **提升效率**：加速開發流程，專注於創意和邏輯而非語法細節
- **實用導向**：強調功能實現和快速原型開發
