# 反應力測試遊戲 - Prompt 參考

> **重要提示**：本遊戲使用 HTML、CSS 和 JavaScript 開發，**不需要安裝任何環境或套件**，直接在瀏覽器中打開 HTML 檔案即可運行。

## 遊戲簡介
反應力測試遊戲是一個測試玩家反應速度的遊戲。玩家需要在看到特定訊號時快速做出反應（例如按下按鍵或點擊滑鼠）。

## 迭代階段 1：基礎版本

### Prompt 範例
```
請幫我建立一個反應力測試遊戲，使用 HTML、CSS 和 JavaScript。
遊戲需要能在瀏覽器中直接運行，不需要安裝任何額外的環境或套件。

基本需求：
1. 建立一個 HTML 檔案，包含一個全螢幕或固定大小的遊戲區域
2. 使用 CSS 設計等待畫面（例如「準備好後按空白鍵」）
3. 使用 JavaScript 處理鍵盤事件，按下空白鍵後，隨機等待一段時間（例如 1-5 秒）
4. 等待時間結束後，使用 JavaScript 改變畫面顏色或顯示「現在！」的提示
5. 玩家需要盡快按下空白鍵（或點擊滑鼠）
6. 使用 JavaScript 的 Date 物件或 performance.now() 計算反應時間（毫秒）
7. 顯示反應時間，可以按 R 鍵或按鈕重新測試

請使用清晰的程式碼結構，並加上註解說明。
```

## 迭代階段 2：多種測試模式

### Prompt 範例
```
在現有的反應力測試遊戲基礎上，請增加以下功能：

1. 多種測試模式：
   - 視覺反應測試：畫面變色時按下按鍵
   - 聽覺反應測試：聽到聲音時按下按鍵
   - 顏色反應測試：顯示特定顏色時按下對應的按鍵

2. 測試結果記錄：
   - 記錄最近 10 次測試的反應時間
   - 計算平均反應時間
   - 顯示最快和最慢的反應時間

3. 統計資訊顯示：
   - 在畫面上顯示測試次數
   - 顯示平均反應時間
   - 顯示最佳反應時間
```

## 迭代階段 3：難度調整與挑戰模式

### Prompt 範例
```
請在反應力測試遊戲中增加難度調整和挑戰模式：

1. 難度選擇：
   - 簡單：等待時間較長（3-7 秒），提示明顯
   - 中等：等待時間中等（1-5 秒），提示正常
   - 困難：等待時間較短（0.5-3 秒），提示較不明顯

2. 挑戰模式：
   - 連續測試模式：連續進行 5 次測試，計算平均反應時間
   - 極限模式：反應時間超過某個閾值（如 500ms）則失敗
   - 時間挑戰：在 30 秒內完成盡可能多的測試

3. 反應時間評級：
   - 根據反應時間給予評級（例如：極快、快、普通、慢）
   - 顯示評級和對應的反應時間範圍
```

## 迭代階段 4：視覺效果與音效

### Prompt 範例
```
請為反應力測試遊戲增加視覺效果和音效：

1. 視覺效果：

2. 音效：
   - 準備音效

   - 背景音樂（可選）

3. 畫面優化：
   - 更清晰的提示設計
   - 反應時間的視覺化顯示（例如：進度條、顏色變化）
   - 統計圖表（可選：反應時間的折線圖）
```

## 迭代階段 5：資料記錄與分析

### Prompt 範例
```
請在反應力測試遊戲中增加資料記錄與分析功能：

1. 歷史記錄：
   - 儲存每次測試的詳細資料（時間戳記、反應時間、模式等）
   - 使用 JSON 或 CSV 格式儲存

2. 資料分析：
   - 顯示反應時間的趨勢圖（折線圖）
   - 顯示反應時間的分佈圖（直方圖）
   - 計算統計數據（平均值、中位數、標準差）

3. 個人最佳記錄：
   - 記錄每種模式的最佳反應時間
   - 記錄每日最佳反應時間
   - 顯示進步趨勢

4. 匯出功能：
   - 可以將測試資料匯出為 CSV 或 JSON 檔案（使用 JavaScript 的 Blob 和下載功能）
   - 方便進一步分析或分享
```

## 技術建議

- **HTML + CSS + JavaScript**：不需要安裝任何環境，直接在瀏覽器中打開 HTML 檔案即可運行
- 使用 **performance.now()** 或 **Date** 物件來精確測量時間
- 使用 **localStorage** 來儲存測試記錄
- 使用 **CSS 動畫** 來實現視覺效果
- 使用 **Chart.js**（可選）來繪製統計圖表，或使用 Canvas 自行繪製

## 學習重點

1. **階段1**：學習時間測量、隨機數生成、事件處理
2. **階段2**：學習多種模式的設計、資料收集與統計
3. **階段3**：學習難度系統、挑戰模式設計
4. **階段4**：學習視覺效果、音效整合
5. **階段5**：學習資料持久化、資料視覺化、統計分析
