# 貪食蛇遊戲 - Prompt 參考

> **重要提示**：本遊戲使用 HTML、CSS 和 JavaScript 開發，**不需要安裝任何環境或套件**，直接在瀏覽器中打開 HTML 檔案即可運行。

## 遊戲簡介
貪食蛇（Snake）是一個經典的街機遊戲。玩家控制一條蛇移動，吃食物來成長，但要避免撞到牆壁或自己的身體。

## 迭代階段 1：基礎版本

### Prompt 範例
```
請幫我建立一個貪食蛇遊戲，使用 HTML、CSS 和 JavaScript。
遊戲需要能在瀏覽器中直接運行，不需要安裝任何額外的環境或套件。

基本需求：
1. 建立一個 HTML 檔案，包含一個遊戲畫布區域（例如 800x600 像素）
2. 使用 HTML5 Canvas 來繪製遊戲畫面
3. 建立一條蛇，初始長度為 3-4 節，使用陣列來儲存蛇的身體座標
4. 使用 JavaScript 處理鍵盤事件，蛇可以用方向鍵（上下左右）控制移動方向
5. 在隨機位置生成食物
6. 當蛇頭碰到食物時：
   - 蛇的長度增加一節（在陣列中加入新的座標）
   - 在隨機位置生成新的食物
7. 遊戲結束條件：
   - 蛇頭碰到牆壁
   - 蛇頭碰到自己的身體
8. 遊戲結束時顯示「遊戲結束」訊息

請使用清晰的程式碼結構，並加上註解說明。使用 requestAnimationFrame 來實現遊戲迴圈。
```

## 迭代階段 2：增加計分與生命系統

### Prompt 範例
```
在現有的貪食蛇遊戲基礎上，請增加以下功能：

1. 計分系統：
   - 每吃一個食物得 10 分
   - 在畫面上顯示當前分數
   - 遊戲結束時顯示最終分數

2. 生命系統（可選）：
   - 玩家有 3 條生命
   - 撞到牆壁或身體時失去一條生命
   - 生命歸零時遊戲結束
   - 在畫面上顯示剩餘生命數

3. 遊戲狀態管理：
   - 開始畫面（按空白鍵開始）
   - 遊戲暫停功能（按 P 暫停/繼續）
   - 遊戲結束畫面（顯示分數，按 R 重新開始）

4. 速度調整：
   - 隨著分數增加，蛇的移動速度逐漸加快
   - 或每吃 5 個食物速度增加一次
```

## 迭代階段 3：增加特殊食物與障礙物

### Prompt 範例
```
請在貪食蛇遊戲中增加特殊食物和障礙物：

1. 特殊食物類型：
   - 普通食物：+1 節，+10 分
   - 金色食物：+2 節，+30 分（較少出現）
   - 減速食物：+1 節，+10 分，且蛇的速度暫時減慢
   - 加速食物：+1 節，+10 分，且蛇的速度暫時加快

2. 障礙物：
   - 在遊戲區域中隨機生成障礙物（牆壁）
   - 蛇碰到障礙物時遊戲結束
   - 障礙物可以用不同顏色或圖案表示

3. 特殊效果持續時間：
   - 減速/加速效果有時間限制（例如 5 秒）
   - 在畫面上顯示效果剩餘時間

4. 食物生成規則：
   - 確保食物不會生成在障礙物上
   - 確保食物不會生成在蛇的身體上
```

## 迭代階段 4：關卡系統與難度調整

### Prompt 範例
```
請在貪食蛇遊戲中增加關卡系統和難度調整：

1. 關卡系統：
   - 多個關卡，每個關卡有不同的障礙物配置
   - 完成關卡的條件：達到目標分數或吃到足夠的食物
   - 完成一關後自動進入下一關

2. 難度選擇：
   - 簡單：速度較慢，障礙物較少
   - 中等：速度正常，障礙物適中
   - 困難：速度較快，障礙物較多

3. 關卡設計：
   - 每個關卡有不同的障礙物排列
   - 關卡難度逐漸增加
   - 顯示當前關卡編號和進度

4. 通關獎勵：
   - 完成關卡時給予額外分數
   - 顯示關卡完成訊息
```

## 迭代階段 5：視覺效果、音效與記錄系統

### Prompt 範例
```
請為貪食蛇遊戲增加視覺效果、音效和記錄系統：

1. 視覺效果：
   - 蛇移動的平滑動畫
   - 吃食物時的動畫效果
   - 遊戲結束時的動畫
   - 背景漸層或圖案
   - 蛇和食物的精美設計

2. 音效：
   - 吃食物音效
   - 吃到特殊食物音效
   - 撞到障礙物音效
   - 遊戲結束音效
   - 背景音樂（可選）

3. 記錄系統：
   - 記錄最高分數
   - 記錄最長蛇的長度
   - 記錄最佳關卡進度
   - 使用瀏覽器的 localStorage 來儲存記錄

4. 統計資訊：
   - 總遊戲次數
   - 平均分數
   - 總共吃過的食物數量
   - 顯示在遊戲結束畫面或主選單
```

## 技術建議

- **HTML + CSS + JavaScript + Canvas**：不需要安裝任何環境，直接在瀏覽器中打開 HTML 檔案即可運行
- 使用 **HTML5 Canvas** 來繪製遊戲畫面
- 使用 **requestAnimationFrame** 來實現流暢的遊戲動畫迴圈
- 使用 **陣列** 來儲存蛇的身體座標
- 使用基本的座標計算來處理碰撞檢測
- 使用 **localStorage** 來儲存遊戲記錄

## 學習重點

1. **階段1**：學習座標系統、列表操作（蛇的身體）、碰撞檢測、隨機生成
2. **階段2**：學習遊戲狀態管理、計分系統、速度控制
3. **階段3**：學習物件導向設計（不同食物類型）、效果系統、條件生成
4. **階段4**：學習關卡設計、難度系統、進度追蹤
5. **階段5**：學習視覺效果、音效整合、資料持久化、統計分析
