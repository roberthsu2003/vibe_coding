# 辦公室工作場景 - 迭代 Prompt 教學

歡迎來到辦公室工作場景的迭代 Prompt 教學！本資料夾專門為辦公室人員設計，幫助您利用 AI 工具（如 Cursor、Claude、ChatGPT）透過迭代的方式，逐步提升工作效率。

## 🎯 適用對象

本教學適合以下工作類型的辦公室人員：
- **Email 寫作**：需要撰寫商務郵件、客戶溝通、內部通知
- **企劃撰寫**：專案企劃、活動規劃、提案文件
- **統計分析**：數據整理、報表製作、趨勢分析
- **行銷內容**：社群貼文、產品文案、行銷素材

## 📚 迭代方式的核心概念

迭代方式的核心是：**從簡單開始，逐步完善**

### 為什麼要用迭代方式？

1. **降低心理負擔**：不需要一次就寫出完美的內容
2. **快速看到成果**：先有基本版本，再逐步優化
3. **學習效果好**：透過多次修正，理解如何與 AI 有效溝通
4. **符合實際工作流程**：真實工作中也是先有初稿，再逐步修改

### 迭代流程

```
階段 1：基礎版本 → 階段 2：增加細節 → 階段 3：優化表達 
→ 階段 4：專業化 → 階段 5：個人化
```

## 🚀 如何使用

1. **選擇您的工作場景**：Email、企劃、統計、或行銷
2. **打開對應的 README.md**：查看該場景的迭代階段
3. **從階段 1 開始**：複製第一個 Prompt 給 AI
4. **逐步迭代**：完成一個階段後，再使用下一個階段的 Prompt
5. **根據需求調整**：每個 Prompt 都可以根據您的實際需求修改

## 💡 使用建議

### 初學者建議

1. **從 Email 開始**：Email 是最日常的工作，容易上手
2. **一次一個階段**：不要跳過階段，每個階段都有學習價值
3. **實際應用**：用真實的工作需求來練習
4. **保留迭代過程**：觀察 AI 如何根據您的指示逐步改進

### 進階使用者建議

1. **組合使用**：可以將不同場景的 Prompt 組合使用
2. **建立模板**：將常用的 Prompt 整理成自己的模板庫
3. **客製化調整**：根據公司文化和個人風格調整 Prompt
4. **分享與改進**：與同事分享有效的 Prompt，共同改進

## 🔄 迭代開發流程

每個工作場景都分為 5 個迭代階段：

1. **階段 1：基礎版本** - 產生基本內容框架
2. **階段 2：增加細節** - 補充具體資訊和細節
3. **階段 3：優化表達** - 改善語言和結構
4. **階段 4：專業化** - 加入專業術語和格式
5. **階段 5：個人化** - 調整語氣和風格，符合個人/公司需求

## 💬 與 AI 協作的技巧

### 1. 提供足夠的上下文

**不好的 Prompt**：
```
幫我寫一封 email
```

**好的 Prompt**：
```
幫我寫一封 email，目的是邀請客戶參加下週的產品發表會。
客戶是科技公司的採購經理，我們之前有過一次愉快的合作經驗。
語氣要專業但友善，需要包含時間、地點、議程等資訊。
```

### 2. 明確指出需要改進的地方

**不好的回饋**：
```
這個不好，重寫
```

**好的回饋**：
```
這封 email 的語氣太正式了，我希望更親切一些。
另外，請在開頭加入一些個人化的問候，例如提到上次會議的內容。
```

### 3. 逐步迭代，不要一次要求太多

**不好的方式**：
```
幫我寫一封完美的 email，要專業、親切、有說服力、包含所有細節...
```

**好的方式**：
```
階段 1：先寫基本架構和主要內容
階段 2：加入具體的時間、地點等細節
階段 3：調整語氣，讓它更親切
階段 4：加入個人化的元素
```
