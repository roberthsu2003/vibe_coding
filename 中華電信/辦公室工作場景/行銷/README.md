# 行銷內容 - 迭代 Prompt 參考

> **重要提示**：本教學使用迭代方式，從簡單的文案初稿開始，逐步完善成專業的行銷內容。

## 使用情境

行銷內容撰寫是辦公室常見的工作，包括：
- 社群媒體貼文
- 產品文案
- 行銷活動企劃
- 廣告文案
- 內容行銷

透過迭代方式，您可以：
- 快速產生文案初稿
- 逐步優化表達和吸引力
- 針對不同平台調整內容
- 加入 CTA（行動呼籲）
- 提升轉換率

## 迭代階段 1：基礎版本

### 目標
產生基本的行銷文案初稿。

### Prompt 範例

#### 情境 A：產品發布貼文

```
請幫我寫一篇產品發布的社群媒體貼文：

1. 產品名稱：[產品名稱]
2. 產品特色：[主要特色，例如：功能、優勢]
3. 目標客群：[要賣給誰]
4. 發布平台：[Facebook / Instagram / LinkedIn 等]
5. 主要訊息：[想要傳達的主要訊息]

請用簡潔有力的方式呈現，先建立基本框架即可。
```

#### 情境 B：活動宣傳

```
請幫我寫一篇活動宣傳的貼文：

1. 活動名稱：[活動名稱]
2. 活動內容：[活動要做什麼]
3. 活動時間和地點：[基本資訊]
4. 目標對象：[誰會參加]
5. 報名方式：[如何報名]

請用吸引人的方式呈現，先建立基本框架即可。
```

#### 情境 C：產品促銷

```
請幫我寫一篇產品促銷的貼文：

1. 產品名稱：[產品名稱]
2. 促銷內容：[折扣、優惠等]
3. 促銷時間：[活動時間]
4. 促銷原因：[為什麼有這個促銷]
5. 購買方式：[如何購買]

請用吸引人的方式呈現，先建立基本框架即可。
```

## 迭代階段 2：增加細節

### 目標
在基礎版本上加入更多吸引人的細節和資訊。

### Prompt 範例

#### 延續情境 A：產品發布貼文

```
請在剛才的產品發布貼文基礎上，增加以下細節：

1. 產品詳細特色：
   - 列出 3-5 個主要功能
   - 說明每個功能的優勢
   - 加入具體的使用場景

2. 目標客群細節：
   - 更詳細描述目標客群
   - 說明產品如何解決他們的問題
   - 加入情感連結

3. 價值主張：
   - 說明產品的獨特價值
   - 與競爭對手的差異
   - 為什麼現在就要擁有

4. 社會證明：
   - 如果有測試用戶的好評，加入引用
   - 如果有數據支持，加入數字
   - 如果有媒體報導，加入連結

5. 視覺元素說明：
   - 建議需要什麼圖片或影片
   - 建議視覺風格
   - 建議色彩和設計元素

請讓貼文更豐富、更有說服力。
```

#### 延續情境 B：活動宣傳

```
請在剛才的活動宣傳貼文基礎上，增加以下細節：

1. 活動詳細內容：
   - 活動的具體流程
   - 活動的亮點和特色
   - 參與者可以獲得什麼

2. 講師或嘉賓資訊：
   - 如果有講師，介紹講師背景
   - 如果有嘉賓，介紹嘉賓特色
   - 為什麼值得參加

3. 活動價值：
   - 參與者可以學到什麼
   - 可以認識什麼人
   - 可以獲得什麼資源

4. 限時優惠：
   - 早鳥優惠（如果有）
   - 限時折扣
   - 特別福利

5. 視覺元素說明：
   - 建議需要什麼視覺素材
   - 建議設計風格
   - 建議吸引人的視覺元素

請讓貼文更吸引人、更有價值感。
```

## 迭代階段 3：優化表達

### 目標
改善語言表達，讓文案更吸引人、更有說服力。

### Prompt 範例

#### 延續情境 A：產品發布貼文

```
請優化剛才的產品發布貼文，讓它：

1. 開頭更吸引人：
   - 用問題、故事、或驚人的事實開頭
   - 抓住讀者的注意力
   - 不要用「我們很高興宣布」這種老套開頭

2. 語言更有力：
   - 使用動詞和行動導向的語言
   - 避免被動語態
   - 使用具體的形容詞

3. 結構更清晰：
   - 使用適當的段落和換行
   - 使用 emoji 增加視覺吸引力（如果適合平台）
   - 使用標題和子標題（如果平台支援）

4. 加入情感元素：
   - 觸動讀者的情感
   - 創造渴望和需求
   - 加入故事性

5. 語氣更親近：
   - 用「你」而不是「您」或「客戶」
   - 用對話式的語氣
   - 讓讀者感覺被理解

請讓這篇貼文更吸引人、更有說服力。
```

#### 延續情境 B：活動宣傳

```
請優化剛才的活動宣傳貼文，讓它：

1. 開頭更有吸引力：
   - 用問題或挑戰開頭
   - 用故事或案例開頭
   - 用驚人的數據或事實開頭

2. 創造緊迫感：
   - 強調限時或限量
   - 強調錯過的機會成本
   - 創造 FOMO（錯失恐懼症）

3. 加入互動元素：
   - 提出問題讓讀者思考
   - 鼓勵讀者留言或分享
   - 創造對話的機會

4. 語言更生動：
   - 使用具體的場景描述
   - 使用感官語言（視覺、聽覺、觸覺等）
   - 使用比喻和類比

5. 結構更易讀：
   - 使用短段落
   - 使用條列式呈現重點
   - 使用適當的換行和空白

請讓這篇貼文更吸引人、更有行動力。
```

## 迭代階段 4：專業化

### 目標
加入專業行銷元素、CTA、平台優化，符合專業行銷標準。

### Prompt 範例

#### 延續情境 A：產品發布貼文

```
請將剛才的產品發布貼文專業化：

1. 加入明確的 CTA（行動呼籲）：
   - 告訴讀者下一步要做什麼
   - 使用行動導向的動詞（立即、現在、馬上）
   - 提供清晰的連結或按鈕文字

2. 平台優化：
   - 根據平台特性調整內容長度
   - 根據平台特性調整格式
   - 根據平台特性調整語氣
   - 加入適合的 hashtag（如果平台支援）

3. 加入追蹤機制：
   - 使用 UTM 參數追蹤連結
   - 設定轉換目標
   - 說明如何追蹤效果

4. 使用專業行銷術語：
   - 用「價值主張」而不是「優點」
   - 用「目標客群」而不是「客戶」
   - 用「轉換率」而不是「效果」

5. 加入 A/B 測試建議：
   - 建議可以測試的變數
   - 建議不同的版本
   - 說明如何評估效果

請讓這篇貼文符合專業行銷標準。
```

#### 延續情境 B：活動宣傳

```
請將剛才的活動宣傳貼文專業化：

1. 加入多層次的 CTA：
   - 主要 CTA：立即報名
   - 次要 CTA：了解更多、分享給朋友
   - 每個 CTA 都有明確的行動指引

2. 平台優化：
   - Facebook：適合較長的內容，加入互動問題
   - Instagram：適合視覺化，強調圖片和影片
   - LinkedIn：適合專業內容，強調價值和學習
   - 根據不同平台調整內容

3. 加入行銷漏斗思考：
   - 意識階段：讓目標對象知道活動
   - 興趣階段：讓他們對活動感興趣
   - 決策階段：讓他們決定參加
   - 行動階段：讓他們實際報名

4. 使用專業行銷框架：
   - AIDA（注意、興趣、渴望、行動）
   - PAS（問題、激化、解決）
   - 根據適合的框架調整內容

5. 加入追蹤和優化：
   - 設定追蹤指標（點擊率、轉換率等）
   - 說明如何優化內容
   - 建議後續的優化方向

請讓這篇貼文符合專業行銷標準。
```

## 迭代階段 5：個人化

### 目標
根據品牌風格、目標客群、實際效果進行最後調整。

### Prompt 範例

#### 延續情境 A：產品發布貼文

```
請將剛才的產品發布貼文個人化：

1. 根據品牌風格調整：
   - 如果品牌較正式，保持專業但親切
   - 如果品牌較輕鬆，可以更活潑有趣
   - 符合品牌的語調和風格指南

2. 根據目標客群調整：
   - 如果目標客群是年輕人，使用更活潑的語言和 emoji
   - 如果目標客群是專業人士，使用更專業的語言
   - 如果目標客群是特定族群，使用他們熟悉的語言和參考

3. 根據平台文化調整：
   - Facebook：較正式，適合長內容
   - Instagram：較視覺化，適合短內容和 emoji
   - LinkedIn：較專業，適合專業內容
   - Twitter：較簡潔，適合短內容和話題

4. 加入個人或品牌特色：
   - 展現品牌的獨特個性
   - 加入品牌的價值觀
   - 展現對產品的熱情

5. 考慮實際執行：
   - 根據實際預算調整內容
   - 根據實際資源調整視覺需求
   - 考慮實際的追蹤和優化能力

請讓這篇貼文既專業又符合品牌和目標客群。
```

#### 延續情境 B：活動宣傳

```
請將剛才的活動宣傳貼文個人化：

1. 根據品牌定位調整：
   - 如果品牌定位是專業，強調專業價值
   - 如果品牌定位是創新，強調創新元素
   - 如果品牌定位是親民，使用更親切的語言

2. 根據目標客群調整：
   - 深入了解目標客群的需求和痛點
   - 使用他們熟悉的語言和參考
   - 觸動他們的情感和動機

3. 根據活動特色調整：
   - 強調活動的獨特之處
   - 展現活動的價值和意義
   - 創造對活動的期待

4. 加入情感連結：
   - 不只是資訊傳達，要創造情感連結
   - 讓讀者感受到活動的價值
   - 創造參與的渴望

5. 考慮實際效果：
   - 根據過去的經驗調整內容
   - 根據目標設定調整重點
   - 考慮實際的轉換目標

請讓這篇貼文既有吸引力又符合實際需求。
```

1. 情況說明：[描述負面情況]

5. 未來承諾：說明我們會如何避免類似問題

請用專業、誠懇、積極的語氣，展現我們對問題的重視和解決的決心。
```

### 情境 2：限時促銷

```
請幫我寫一篇限時促銷的貼文：

1. 促銷內容：[說明促銷內容]

3. 促銷原因：[說明為什麼有這個促銷]
4. 價值強調：[強調產品的價值和優惠的吸引力]
5. 行動呼籲：明確告訴讀者要立即行動

請用吸引人、有緊迫感的方式呈現，讓讀者想要立即行動。

```

3. 內容：實用的知識或技巧

5. 行動呼籲：鼓勵讀者了解更多或採取行動

請用有價值、有教育意義的方式呈現，建立信任和權威。
```

## 使用技巧

### 1. 提供足夠的上下文

**不好的 Prompt**：
```
幫我寫一篇行銷貼文
```

**好的 Prompt**：
```
請幫我寫一篇產品發布的 Facebook 貼文：
- 產品：新的手機 App，主要功能是幫助用戶管理時間
- 目標客群：忙碌的上班族和學生
- 品牌風格：專業但親切，強調效率和實用
- 主要訊息：這個 App 可以幫助用戶更有效地管理時間
- 需要包含：產品特色、使用場景、下載連結
```

### 2. 明確指出需要改進的地方

**不好的回饋**：
```
這個不夠吸引人
```

**好的回饋**：
```
這篇貼文的開頭太普通了，請改成：
1. 用問題開頭，例如「你每天浪費多少時間在滑手機？」
2. 用故事開頭，例如「小明的時間管理故事」
3. 用驚人的事實開頭，例如「平均每人每天浪費 2 小時在無意義的事情上」
另外，請加強情感連結，讓讀者感受到時間管理的重要性。
```

### 3. 逐步迭代，不要一次要求太多

**不好的方式**：
```
幫我寫一篇完美的行銷貼文，要吸引人、有說服力、有 CTA、
符合品牌風格、針對目標客群、還要優化轉換率...
```

**好的方式**：
```
階段 1：先寫基本架構和主要內容
階段 2：加入吸引人的細節和價值主張
階段 3：優化表達和結構
階段 4：加入專業行銷元素和 CTA
階段 5：根據品牌和目標客群進行最後調整
```

1. **從真實需求開始**：用您實際需要寫的行銷內容來練習
2. **保留迭代過程**：觀察 AI 如何根據您的指示逐步改進
3. **建立內容庫**：將常用的行銷內容類型整理成模板
4. **測試和優化**：根據實際效果，調整 Prompt 策略
