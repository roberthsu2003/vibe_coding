# Chatbot 建立

## 課堂規劃（約）

- 定義 Chatbot 角色：**客服問答**（常見問題、語氣、拒答範圍）。
- 準備**知識來源**：簡短 FAQ 文字、或幾段產品說明（勿使用未授權或機密內容）。
- 用系統提示或 RAG（若課程涵蓋）限制回答範圍，避免幻覺；設計「無法回答時」的固定話術。
- 實作對話介面：網頁聊天室或嵌入式（依講義），送出使用者訊息 → 顯示模型回覆。
- 測試邊界案例：無關問題、惡意輸入、超長文字，並迭代 Prompt。

## 作品（產出）

以 AI 建立**客服問答型 Chatbot**
