# 資料整理工具

## 課堂規劃（約）

- 準備一組 **CSV**（可用假資料）：欄位名稱、幾十列即可，避免過大檔案。
- 請 AI 協助理解欄位意義、檢查缺值與異常，並定義「摘要」要回答哪些問題（總計、分類、趨勢）。
- 實作流程：上傳或貼上 CSV 片段 → 模型分析 → 輸出**摘要報告**（Markdown 或 HTML）。
- 迭代 Prompt：要求圖表描述、表格化統計、或依某欄位分組（若工具支援）。
- 留存：範例 CSV、分析結果範本、使用限制說明（列數／token 上限）。

## 作品（產出）

匯入 **CSV**，由 AI 分析並輸出**摘要報告**

## 實作請放此處

- 範例 CSV（可為假資料）、分析結果範本。
- 應用程式碼或 Notebook（依實作形式）。
