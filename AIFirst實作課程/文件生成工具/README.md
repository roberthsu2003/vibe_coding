# 文件生成工具

## 課堂規劃（約）

- 選定一種情境：**會議紀錄**、**Email 草稿**或**簡要報告**（或三者各做迷你版）。
- 設計「輸入範本」：例如會議中有誰發言、待辦事項欄位；Email 的收件人語氣與目的。
- 用 Prompt 約束輸出格式（標題、條列、表格、時間戳），並依實際結果迭代。
- 若做成小工具：單頁輸入框 → 呼叫模型 → 顯示輸出，可複製下載。
- 注意：真實會議內容若含個資，請用**去識別化範例**練習。

## 作品（產出）

以 AI 產生**會議紀錄**、**Email 草稿**或**簡要報告**

## 實作請放此處

- 範例輸入／輸出（可去識別化）。
- 若為網頁或腳本專案，請將程式碼置於此資料夾。
