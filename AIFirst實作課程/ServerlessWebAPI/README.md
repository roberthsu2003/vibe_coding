# Serverless（無伺服器）Web API

## 課堂規劃（約）

- 建立 **Gemini API** 金鑰（或依課程使用 Google AI Studio 串接方式），**絕不**將金鑰寫入公開儲存庫。
- 簡介 **無伺服器（Serverless）**：託管方代管伺服器，依請求執行函式，無需自管機器。
- 在指定託管環境（如 Cloud Functions、Edge Functions 等，依講義）建立一個 **HTTP 端點**，後端呼叫 Gemini API。
- 練習用環境變數保存金鑰、區分本機與正式環境；部署後以瀏覽器或 curl 測試。
- 整理部署步驟與錯誤排查筆記（常見：CORS、金鑰權限、區域）。

## 作品（產出）

以託管環境串接 **Gemini API**，練習無伺服器部署概念
