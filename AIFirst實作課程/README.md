# AI First 實作課程

本資料夾對應主文件中的 **AI First 課程規劃（Google AI Studio）**：每一主題一個子資料夾，請將該主題的實作程式、截圖、備註與延伸練習放在對應目錄中。

**各主題資料夾內的 `README.md` 說明什麼？**

- **課堂規劃（約）**：該主題建議的教學流程、練習步驟與注意事項（可依班級進度刪減）。

- **實作請放此處**：建議您把程式、截圖、連結等實作產出放在同一資料夾內，方便日後整理與展示。
