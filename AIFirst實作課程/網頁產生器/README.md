# 網頁產生器

## 課堂規劃（約）

- 決定頁面目的：**個人介紹**、**作品集列表**或兩者合一；列出必備區塊（標題、簡介、作品縮圖、聯絡方式等）。
- 用 Prompt 描述風格（簡潔／深色／雜誌排版等）與**響應式**需求，請 AI 產生初版 HTML／CSS。
- 多輪迭代：替換文案、調整版面、加上導覽列或錨點；避免一次要求過多改動。
- 產出靜態檔案，於本機與手機寬度預覽檢查；必要時請 AI 協助修正 CSS。
- 將成品部署至 GitHub Pages 或自訂網址（依課程進度）。
