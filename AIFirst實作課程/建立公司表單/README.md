# 建立公司表單

## 課堂規劃（約）

- 盤點表單欄位：姓名、聯絡方式、選項題等；區分必填／選填與格式驗證（Email、電話）。
- 用 AI 產出**表單前端**（HTML 表單或表單建置工具產出），並考慮行動裝置可填寫。
- 設計「提交後」流程：資料寫入 **Google 試算表**、**表單後端**、或通知信（依講義）。
- **自動整理**：提交結果匯總成表格、或每日摘要信；練習用假資料測試。
- 法遵與隱私：個資最小化、告知用途；**勿**在公開 repo 放真實個資測試檔。

## 作品（產出）

以 AI 建立**報名表單**，並**自動整理提交結果**

## 實作請放此處

- 表單頁面、後端或試算表／資料庫串接說明（依實作）。
- 測試資料請勿使用真實個資。
