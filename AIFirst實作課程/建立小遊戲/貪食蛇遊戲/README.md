# 貪食蛇遊戲 - Prompt 參考

> **重要提示**：本專案使用 Vite、React 和 TypeScript 開發。建議學員先安裝 Node.js 環境（並使用 `npm create vite@latest` 建立專案），或直接使用線上編輯器（如 CodeSandbox、StackBlitz）來執行專案。

## 遊戲簡介
貪食蛇（Snake）是一個經典的街機遊戲。玩家控制一條蛇移動，吃食物來成長，但要避免撞到牆壁或自己的身體。

## 迭代階段 1：基礎版本

### Prompt 範例
```
請幫我建立一個貪食蛇遊戲，使用 Vite、React 和 TypeScript。

基本需求：
1. 建立一個 React 元件，包含一個遊戲畫布區域（例如 800x600 像素）
2. 在 React 中使用 HTML5 Canvas（搭配 useRef）來繪製遊戲畫面，或使用 CSS Grid 實作網格系統
3. 建立一條蛇，初始長度為 3-4 節，使用狀態 (State) 與陣列來儲存蛇的身體座標
4. 使用 React 的 useEffect 處理鍵盤事件，蛇可以用方向鍵（上下左右）控制移動方向
5. 在隨機位置生成食物
6. 當蛇頭碰到食物時：
   - 蛇的長度增加一節（在陣列中加入新的座標）
   - 在隨機位置生成新的食物
7. 遊戲結束條件：
   - 蛇頭碰到牆壁
   - 蛇頭碰到自己的身體
8. 遊戲結束時顯示「遊戲結束」訊息

請使用清晰的元件結構，並加上 TypeScript 型別與註解說明。使用 requestAnimationFrame 或 setInterval 在 useEffect 中實現遊戲迴圈。
```

## 迭代階段 3：增加特殊食物與障礙物

### Prompt 範例
```
請在貪食蛇遊戲中增加特殊食物和障礙物，並定義相關的 TypeScript 型別 (Interface)：

1. 特殊食物類型：
   - 普通食物：+1 節，+10 分
   - 金色食物：+2 節，+30 分（較少出現）
   - 減速食物：+1 節，+10 分，且蛇的速度暫時減慢
   - 加速食物：+1 節，+10 分，且蛇的速度暫時加快

2. 障礙物：
   - 在遊戲區域中隨機生成障礙物（牆壁座標陣列）
   - 蛇碰到障礙物時遊戲結束
   - 障礙物可以用不同顏色或圖片繪製

3. 特殊效果持續時間：
   - 減速/加速效果有時間限制（例如 5 秒）
   - 實作計時器並在畫面上顯示效果剩餘時間

4. 食物生成規則：
   - 確保食物不會生成在障礙物上
   - 確保食物不會生成在蛇的身體上
```

## 技術建議

- **環境建置**：建議使用 `npm create vite@latest game -- --template react-ts` 快速建立專案。
- **渲染方式**：可以使用 **HTML5 Canvas** 搭配 `useRef` 來繪製高效能的遊戲畫面，或是直接使用 React 渲染 **CSS Grid** 網格系統。
- **狀態管理**：使用 `useState`、`useReducer` 或 `useRef` 來管理遊戲狀態（如蛇的位置陣列、食物座標、分數、生命值）。注意：頻繁更新的遊戲座標建議使用 `useRef` 以避免不必要的重新渲染，或結合 `requestAnimationFrame` 進行最佳化。
- **副作用處理**：使用 `useEffect` 來綁定鍵盤事件監聽器，以及處理遊戲迴圈的計時器（`setInterval` 或 `requestAnimationFrame`）。
- **資料持久化**：使用 `localStorage` 來儲存遊戲記錄。

## 學習重點

1. **階段1**：學習 React `useRef` 與 Canvas 的整合、陣列操作（蛇的身體推進）、基礎 TypeScript 型別
2. **階段2**：學習 React 狀態管理 (`useState`)、遊戲迴圈實作與元件生命週期 (`useEffect` cleanup)
3. **階段3**：學習 TypeScript Interface 定義（不同食物與效果型別）、條件生成與碰撞邏輯
4. **階段4**：學習配置驅動開發 (Configuration-driven development)、關卡切換邏輯
5. **階段5**：學習 Audio API 整合、自訂 Hook 封裝 (如 `useLocalStorage`)、進階動畫與效能最佳化
