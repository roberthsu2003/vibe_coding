# 建立小遊戲

## 課堂規劃（約）

- 先**口頭或條列寫出遊戲規則**（勝負條件、輸入方式、回饋訊息），再交給 AI 產生初版。
- 擇一或兩者：**猜數字**（範圍、次數提示）、**反應力遊戲**（計分、計時）— 依時間調整深度。
- 在 Google AI Studio 中迭代：修正邏輯 bug、調整 UI 文案與版面、補上重新開始／說明。
- 產出可執行的**單頁或少量檔案**（HTML／CSS／JS），並在瀏覽器實際遊玩測試。
- 將可分享連結或檔案納入作品集（可與 GitHub Pages 搭配）。

## 作品（產出）

以 AI 產生網頁小遊戲（**猜數字**或**反應力遊戲**擇一或兩者）

## 實作請放此處

- 遊戲相關 HTML／CSS／JS 或單一頁面專案。
- 可簡述規則與如何在本機或 GitHub Pages 開啟。
