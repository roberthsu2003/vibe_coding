# 娛樂稅VS納保法記憶翻牌
16張暗光鳥卡牌,點擊翻牌,翻出2張相同的牌即完成配對,有15次挑戰機會,於時間內配對完成即過關!

## Prompt

# 任務
請建立一個「記憶翻牌遊戲」的單頁應用程式，使用 Vite、React、TypeScript 進行開發。

# 響應式與設備支援需求
1. 支援桌機。
2. 支援手機直式（Portrait）顯示。
3. 若為橫式（Landscape）顯示時，出現警告提示（例如：「此遊戲僅支援直式模式，請將裝置轉為直式」），並遮蔽或隱藏遊戲主畫面。

# 遊戲主題
娛樂稅 VS 納保法 記憶翻牌遊戲

# 遊戲規則
1. 共 16 張卡牌（8 對）
2. 每張卡牌代表：
   - 娛樂稅課稅項目
   - 納保法相關概念
3. 玩家每次可以翻開 2 張卡牌
4. 若兩張相同 → 配對成功
5. 配對成功時：
   - 增加 15 秒時間
6. 玩家只有 15 次翻牌機會（每翻 2 張算 1 次）
7. 必須在時間內完成所有配對才算過關
8. 若時間歸零或機會用完 → 遊戲失敗

# 技術需求

## 框架與工具
- Vite
- React
- TypeScript

## CSS / 樣式
- 使用 flex 或 grid 排版
- 卡牌翻轉動畫
- 綠色遊戲風格背景
- 按鈕為橘色
- CSS Media Query: 使用 `@media (orientation: landscape)` 來偵測橫向模式，並顯示警告畫面。

## React 程式設計 (Hooks)
需要包含：

### 遊戲狀態 (State)
- 剩餘時間
- 剩餘機會
- 已配對數量
- 是否正在翻牌中

### 功能
1. 初始化卡牌
2. 隨機排序
3. 點擊翻牌
4. 判斷配對
5. 計算機會
6. 計時器 (利用 `useEffect` 等)
7. 配對成功加 15 秒
8. 顯示成功或失敗畫面
