# 使用牌照稅迷宮大冒險(license-tax-maze)

## 遊戲簡介
1.迷宮內共有4道謎題，全部解答後出口就會出現。
2.方向鍵控制車輛移動，滑鼠點擊問題選項。
★幫助暗光鳥解答謎題，讓他們順利回家吧！

# 角色與任務
你是一位專業的 React 前端開發工程師，精通 TypeScript 與 Vite 專案架構。請幫我開發一款名為「使用牌照稅迷宮大冒險」的 Web 互動小遊戲。


# 技術棧要求
- 核心框架：React 18+ (Functional Components + Hooks)
- 開發環境：Vite
- 語言：TypeScript (需定義嚴謹的 Props 與 State 介面)
- 樣式：可以使用純 CSS 或 CSS Modules，請確保畫面具有 RWD 響應式設計。


# 核心遊戲機制
1. **迷宮探索**：玩家需操控一台綠色汽車在迷宮中尋找謎題。迷宮帶有「探照燈（視野限制）」效果，只有車輛周圍區域是亮的，其餘迷宮隱藏在黑暗中（可透過 Canvas API 或 CSS 遮罩實作）。
2. **解謎觸發**：迷宮內共有 4 道謎題，當車輛碰到帶有「Q」字樣的標記時，會觸發狀態切換，跳出對應的問答題 Modal。
3. **出口機制**：解開全部 4 道謎題後，畫面上必須彈出提示「您已解開所有謎題！出口已出現於某處。快去尋找吧！」，並在迷宮邊緣生成一扇木門作為出口。
4. **跨平台操作（必備）**：
   - **桌機版**：透過 `useEffect` 監聽鍵盤方向鍵 (`keydown`) 控制車輛移動，使用滑鼠點擊問題選項。
   - **手機版（直排與橫排自適應）**：必須實作螢幕虛擬搖桿 (Virtual Joystick) 或 UI 方向按鈕組來控制車輛，支援觸控。遊戲畫面需能根據手機螢幕動態縮放且不破版。


# 視覺與場景設計 (Component 劃分建議)
1. **`TitleScreen` 組件**：顯示標題「使用牌照稅迷宮大冒險」。說明文字：「暗光鳥們開車開到一座迷宮內，請幫助他們解答各樣租稅謎題，以找到回家的路吧！(共4題)」與「操作說明：方向鍵控制車輛，滑鼠回答問題」。畫面包含一台紅色汽車與「PLAY」按鈕。
2. **`MazeBoard` 組件**：負責渲染迷宮地圖、車輛位置、Q標記、探照燈遮罩以及最終出現的木門。
3. **`QuestionModal` 組件**：採用綠色黑板風格的 UI 顯示題目與選項，左上角標示「新北租稅小教室」。答對時切換至提示文字，右上角包含「X」關閉按鈕。
4. **`VictoryScreen` 組件**：通關畫面，顯示一隻藍色貓頭鷹拿麥克風，對話框文字：「恭喜通關！歡迎至本處官網挑戰其他租稅小遊戲！」。
