# 認識 Google AI Studio

## 課堂規劃（約）

- 建立／登入 **Google AI Studio**，瀏覽建立應用、模型選擇、系統提示（若有）等基本介面。
- 釐清「輸入 → 模型 → 輸出」流程，並嘗試更換模型或參數，觀察差異。
- 練習用自然語言描述需求，再**小步修改 Prompt**，比對前後結果（迭代式調整）。
- 完成一個最小可行作品：**文字產生器**（例如依主題產生段落、改寫語氣、條列重點等）。
- 整理本次使用的 Prompt 要點與失敗／成功經驗，作為後續主題的基礎。

## 作品（產出）

第一個 AI 應用：**文字產生器**
