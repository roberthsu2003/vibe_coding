# 範例說明：雙班競賽計時器

本文件提供一則可在 **Google AI Studio**（或任何支援產生網頁的 AI 工具）練習的 **Prompt 範例**：從一句話需求出發，對照「原始想法 → 潤飾後 Prompt」，示範如何把**目的、對象、畫面與按鈕**說清楚，方便 AI 一次產出可用的單頁應用。

---

## 原始需求（學員可先這樣想）

```
建立2個班級的計時器
目的:為了比賽計時用
```

以上已點出**數量（兩班）**與**情境（比賽）**，但尚未說明「獨立計時／按鈕／畫面形式」，AI 可能各做各的。下一步請用下面**潤飾版**。

---

## 潤飾後 Prompt（建議直接複製使用）

請用 **HTML、CSS、JavaScript** 製作**單一網頁**（可合併在同一個 `.html` 檔，內嵌樣式與腳本），內容為**兩組互相獨立的計時器**，供**兩個班級在比賽或活動中同時計時**使用。

### 功能

1. 畫面上有**兩組計時區塊**（預設標籤可為「班級 1」「班級 2」，或提供輸入框讓使用者自訂班級名稱）。
2. **每一組**皆需有：**開始**、**暫停**、**重設（歸零）** 按鈕；兩組計時**互不干擾**。
3. 時間顯示清楚，建議 **MM:SS**（若需較長競賽可支援 **HH:MM:SS**）；數字**夠大、易讀**，適合教室投影或筆電螢幕分享。
4. 介面簡潔，避免不必要動畫影響閱讀；配色對比足夠。

### 非功能需求

- **不需後端**、不需資料庫；若需記住班級名稱，可用 `localStorage`（選填）。
- 請輸出**完整可貼上即執行**的程式碼，並用三五行中文說明**操作方式**與**注意事項**（例如：重設前是否需確認）。

---

## 精簡版 Prompt（課堂時間有限時）

請做一個網頁：上面**兩個獨立計時器**，給**兩班比賽計時**用；每個都有開始、暫停、重設；字大、好讀；單一 HTML 檔、不需後端。請給完整程式碼。

---

## 教學建議

1. **先 Preview**：確認兩組是否真的獨立、按鈕是否符合預期。  
2. **再迭代**：若字不夠大、沒有暫停、標籤想改成「紅隊／藍隊」等，用**短句**補充請 AI 修改。  
3. **下載**：切換到 **Code 模式**再下載專案（見主 README 說明）。  
4. **分享**：可 **Share** 給同組 Remix，或依課程進度再談 **Publish**。
