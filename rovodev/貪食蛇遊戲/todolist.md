## 📁 建議的檔案結構
```
snake-game/
├── index.html          # 主頁面
├── css/
│   └── style.css       # 樣式文件
├── js/
│   ├── game.js         # 主遊戲邏輯
│   ├── snake.js        # 蛇的類別
│   ├── food.js         # 食物類別
│   └── utils.js        # 工具函數
└── assets/
    ├── images/         # 圖片資源
    └── sounds/         # 音效資源 (可選)
```
