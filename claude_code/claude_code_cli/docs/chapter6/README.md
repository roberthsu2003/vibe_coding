# 第六章：實戰練習（手把手做專案）

## 章節簡介

📌 **強烈推薦！** 本章透過實際專案帶你學習如何使用 Claude Code CLI。我們將建立一個完整的 Express API 專案，包含測試、文檔和自動化部署。

## 專案目標

建立一個**待辦事項 API**，包含：
- RESTful API 設計
- 資料庫操作
- 單元測試
- API 文檔
- 自動化部署腳本
