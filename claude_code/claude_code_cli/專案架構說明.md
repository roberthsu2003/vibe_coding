## 📝 內容特色

### 與 VSCode 版本的差異
1. **重點不同**
   - VSCode 版：圖形介面操作
   - CLI 版：命令列操作、自動化、批次處理

2. **目標讀者**
   - VSCode 版：完全初學者
   - CLI 版：有基礎的開發者、命令列愛好者

3. **實戰專案**
   - VSCode 版：待辦事項 Web 應用
   - CLI 版：Express API + 自動化部署
