# 5.1 功能對照表

Claude Code 有兩種使用方式，讓我們來比較它們的特性：

## 對照表

| 特性 | VSCode 版本 | CLI 版本 |
|------|-------------|----------|
| 操作方式 | 圖形介面點選 | 終端機打指令 |
| 適合對象 | 初學者、視覺導向 | 進階、自動化需求 |
| 代碼整合 | 直接插入編輯器 | 需要手動複製 |
| 學習曲線 | 低（直覺操作） | 中（需懂命令列） |
| 多檔案處理 | 視覺化選擇檔案 | 用指令指定 |
| 自動化腳本 | 較難 | 容易 |
| 檢視修改 | Inline Diff 比對 | 文字輸出 |
| 多工處理 | 支援多個 Session | 單一 Session |
| 快捷鍵 | 豐富的快捷鍵 | 命令列參數 |
| 設定檔 | GUI 設定介面 | 設定檔編輯 |
