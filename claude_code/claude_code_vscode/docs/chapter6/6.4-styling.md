# 6.4 步驟三：美化樣式

現在功能完成了，但看起來有點陽春。讓我們加入 CSS 讓它更美觀！

## 💬 在 Claude Code 輸入：

```
為這個待辦事項頁面加入美觀的 CSS 樣式：
1. 使用柔和的配色
2. 置中顯示
3. 輸入框和按鈕要有現代感
4. 清單項目要清楚易讀
5. 按鈕要有 hover 效果
6. 整體風格簡潔優雅

請把 CSS 放在 <style> 標籤中。
```

📸 *[需要加入操作截圖]*

## Claude 會生成 CSS 樣式：

```html
<style>
    * {
        margin: 0;
        padding: 0;
        box-sizing: border-box;
    }

    body {
        font-family: 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif;
        background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
        min-height: 100vh;
        display: flex;
        flex-direction: column;
        align-items: center;
        justify-content: center;
        padding: 20px;
    }

    h1 {
        color: white;
        font-size: 2.5rem;
        margin-bottom: 30px;
        text-shadow: 2px 2px 4px rgba(0,0,0,0.2);
    }

    .input-container {
        background: white;
        padding: 30px;
        border-radius: 15px;
        box-shadow: 0 10px 30px rgba(0,0,0,0.2);
        margin-bottom: 30px;
        display: flex;
        gap: 10px;
        max-width: 500px;
        width: 100%;
    }

    #todoInput {
        flex: 1;
        padding: 12px 20px;
        border: 2px solid #e0e0e0;
        border-radius: 25px;
        font-size: 16px;
        outline: none;
        transition: border-color 0.3s;
    }

    #todoInput:focus {
        border-color: #667eea;
    }

    #addBtn {
        padding: 12px 30px;
        background: #667eea;
        color: white;
        border: none;
        border-radius: 25px;
        font-size: 16px;
        cursor: pointer;
        transition: all 0.3s;
        font-weight: bold;
    }

    #addBtn:hover {
        background: #5568d3;
        transform: translateY(-2px);
        box-shadow: 0 5px 15px rgba(102, 126, 234, 0.4);
    }

    #todoList {
        list-style: none;
        max-width: 500px;
        width: 100%;
    }

    #todoList li {
        background: white;
        padding: 15px 20px;
        margin-bottom: 10px;
        border-radius: 10px;
        display: flex;
        justify-content: space-between;
        align-items: center;
        box-shadow: 0 3px 10px rgba(0,0,0,0.1);
        animation: slideIn 0.3s ease;
    }

    @keyframes slideIn {
        from {
            opacity: 0;
            transform: translateY(-10px);
        }
        to {
            opacity: 1;
            transform: translateY(0);
        }
    }

    #todoList li span {
        flex: 1;
        font-size: 16px;
        color: #333;
    }

    .delete-btn {
        padding: 8px 20px;
        background: #ff6b6b;
        color: white;
        border: none;
        border-radius: 20px;
        cursor: pointer;
        transition: all 0.3s;
        font-size: 14px;
    }

    .delete-btn:hover {
        background: #ee5a52;
        transform: scale(1.05);
    }
</style>
```

## 將 CSS 加入你的 HTML：

將 `<style>` 標籤放在 `<head>` 區域中
