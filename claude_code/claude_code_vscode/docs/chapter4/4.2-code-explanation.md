# 4.2 代碼解釋

當你看不懂某段程式碼時，Claude 可以幫你解釋得清清楚楚。

## ✏️ 範例：選取一段代碼 → 右鍵 → Ask Claude

### 方法 1：使用右鍵選單

1. 📸 用滑鼠選取程式碼
2. 📸 按右鍵，選擇「Ask Claude」或「Send to Claude Code」
3. 📸 Claude Code 面板會自動開啟
4. 追加你的問題：「這段程式碼在做什麼？」

### 方法 2：直接在 Claude Code 輸入框提問

💬 複製程式碼並輸入：
```
請解釋這段程式碼在做什麼：
[貼上程式碼]
```
