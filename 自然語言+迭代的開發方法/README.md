# 自然語言 ＋ 迭代的開發方法

在 Vibe Coding 裡，你不用先寫規格書，也不用一次想清楚所有細節。只要**用自然語言說出目標**（例如：「開發貪食蛇遊戲」），讓 AI 產出**第一個原型**，再透過**迭代**一步步把應用程式做完善。


## 1. 開發流程：一句話 → 原型 → 迭代完善

### 第一步：用自然語言說出目標

不需要寫技術規格，用平常說話的方式描述你想做的東西就好。

**例如：**
> 「請幫我寫一個貪食蛇遊戲」

### 第二步：AI 產出第一個原型

AI 會根據你的描述，產出一版**可以跑、可以玩**的程式。這一版通常已經具備基本功能：畫布、蛇會動、有食物、吃到會變長。

這一版不必完美，重點是**先有東西可以看、可以操作**。

### 第三步：用迭代讓應用程式更完善

拿到第一版之後，不要一次要求「全部重寫」，而是**一次改一點**，用對話引導 AI 補上你想要的東西。
每完成一輪「描述 → AI 修改 → 你檢視」，應用程式就離你心中的樣子更近一步。這就是**迭代**。
