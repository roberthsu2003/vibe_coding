# 自然語言 ＋ 迭代的開發方法

在 Vibe Coding 裡，你不用先寫規格書，也不用一次想清楚所有細節。只要**用自然語言說出目標**（例如：「開發一個個人記帳本」），讓 AI 產出**第一個原型**，再透過**迭代**一步步把應用程式做完善。


## 1. 開發流程：一句話 → 原型 → 迭代完善

### 第一步：用自然語言說出目標

不需要寫技術規格，用平常說話的方式描述你想做的東西就好。

**例如：**
> 「請幫我寫一個個人記帳網頁，包含收入與支出的紀錄表，以及一個顯示比例的圓餅圖。」

### 第二步：AI 產出第一個原型

AI 會根據你的描述，產出一版**可以跑、可以使用**的程式。這一版通常已經具備基本功能：輸入金額與類別的表單、顯示紀錄的表格，以及基礎的圖表。

這一版不必完美，重點是**先有東西可以看、可以操作**。

### 第三步：用迭代讓應用程式更完善

拿到第一版之後，不要一次要求「全部重寫」，而是**一次改一點**，用對話引導 AI 補上你想要的東西。

**迭代範例：**
1. 「請幫我加入深色模式，並讓介面看起來更現代化。」
2. 「在表格上方增加一個本月總結區域，顯示總收入、總支出與餘額。」
3. 「增加一個篩選功能，讓我可以按月份查看紀錄。」

每完成一輪「描述 → AI 修改 → 你檢視」，應用程式就離你心中的樣子更近一步。這就是**迭代**。
