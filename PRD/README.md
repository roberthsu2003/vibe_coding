# PRD需求文件開發

## 📋 PRD 需求文件開發 - 如何使用

1. 選擇一個你感興趣的遊戲
2. 打開 `PRD/` 資料夾中對應的 README.md 檔案
3. 閱讀 PRD 範例，了解如何撰寫需求文件
4. 可以完全使用範例 PRD，或根據範例修改成自己的需求
5. 將完整的 PRD 提供給 AI，讓 AI 根據需求產生程式碼
6. 測試產生的程式碼，如有問題可修改 PRD 或直接告訴 AI 需要調整的地方

## 🎯 技術說明

- **技術棧**：HTML + CSS + JavaScript
- **環境需求**：**不需要安裝任何環境或套件**
- **執行方式**：直接在瀏覽器中打開 HTML 檔案即可運行
- **資料儲存**：使用瀏覽器的 `localStorage` 來儲存遊戲記錄

## 🎮 遊戲列表

### 1. [記憶翻牌遊戲](./PRD/memory_card/README.md)

**遊戲簡介**：記憶翻牌遊戲（Memory Card Game）是一個考驗記憶力的遊戲。玩家需要翻開卡片，記住卡片的位置，然後找出配對的卡片。

**學習重點**：
- 基本的遊戲迴圈、事件處理、狀態管理
- 資料追蹤與顯示
- 參數化設計與條件邏輯
- 動畫與多媒體處理
- 資料持久化與 localStorage 的使用

**適合程度**：⭐⭐⭐ 中等

---

### 2. [打磚塊遊戲](./PRD/breakout/README.md)

**遊戲簡介**：打磚塊（Breakout）是一個經典的街機遊戲。玩家控制一個平台（paddle）來反彈球，擊破上方的磚塊。目標是清除所有磚塊。

**學習重點**：
- 物理碰撞檢測、向量運算、遊戲迴圈
- 遊戲狀態管理、生命系統設計
- 物件導向設計（不同磚塊類型）、效果系統
- 關卡設計、資料結構（關卡資料儲存）
- 視覺效果實作、音效整合

**適合程度**：⭐⭐⭐⭐ 中高

---

### 3. [反應力測試遊戲](./PRD/reaction_test/README.md)

**遊戲簡介**：反應力測試遊戲是一個測試玩家反應速度的遊戲。玩家需要在看到特定訊號時快速做出反應（例如按下按鍵或點擊滑鼠）。

**學習重點**：
- 時間測量、隨機數生成、事件處理
- 多種模式的設計、資料收集與統計
- 難度系統、挑戰模式設計
- 視覺效果、音效整合
- 資料持久化、資料視覺化、統計分析

**適合程度**：⭐⭐ 簡單

---

### 4. [猜數字遊戲](./PRD/guess_number/README.md)

**遊戲簡介**：猜數字遊戲是一個經典的邏輯推理遊戲。電腦隨機產生一個數字，玩家需要透過提示（太大、太小）來猜出正確的數字。

**學習重點**：
- 隨機數生成、條件判斷、迴圈控制、輸入處理
- 參數化設計、範圍計算、遊戲設定
- GUI 設計、事件處理、視覺化顯示
- 演算法（二分搜尋）、策略分析、教學功能
- 多人遊戲設計、競賽系統、資料比較

**適合程度**：⭐ 最簡單（推薦初學者從這裡開始）

---

### 5. [貪食蛇遊戲](./PRD/snake/README.md)

**遊戲簡介**：貪食蛇（Snake）是一個經典的街機遊戲。玩家控制一條蛇移動，吃食物來成長，但要避免撞到牆壁或自己的身體。

**學習重點**：
- 座標系統、列表操作（蛇的身體）、碰撞檢測、隨機生成
- 遊戲狀態管理、計分系統、速度控制
- 物件導向設計（不同食物類型）、效果系統、條件生成
- 關卡設計、難度系統、進度追蹤
- 視覺效果、音效整合、資料持久化、統計分析

**適合程度**：⭐⭐⭐ 中等
