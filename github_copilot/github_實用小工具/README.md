## MCP自動開啟

### 設定步驟

1. 搜尋關鍵字`mcp`
2.Chat > Mcp:Autostart

![](./images/pic2.png)

## AI Stats(AI統計)

- 統計使用AI的狀態

### 設定步驟

1. 搜尋關鍵字`stats`
2. Editor > Ai Stats:Enabled

![](./images/pic3.png)
![](./images/pic4.png)

## todolist

觀察任務拆解和執行狀況

### 設定步驟

1. 搜尋關鍵字`todo list`
2. Chat> Todo List Tool:Enabled

![](./images/pic5.png)
