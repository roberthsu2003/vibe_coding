### 小遊戲 Prompt 範例
以下範例提供「初次建立專案」與「功能迭代」兩階段的 Prompt 建議，方便學員在 Cursor 中直接使用或調整。

#### 記憶翻牌 Memory Match
- **初始 Prompt**：
  - 「幫我用 HTML、CSS、JavaScript 建立一個記憶翻牌遊戲。畫面有 4x4 的卡片格子，隨機配對圖示。玩家翻開兩張牌時，比對是否相同，相同則保持翻開，不同則 1 秒後蓋回去。左上角顯示翻牌次數，遊戲結束顯示完成時間。」
- **迭代 Prompt**：
  - 「加入開始與重新開始按鈕，重新開始時洗牌並重置計時與翻牌次數。」
  - 「新增難度選擇（3x4、4x4、4x5），並在畫面上顯示目前難度。」
  - 「完成遊戲後顯示一個彈出視窗，記錄最佳時間，使用 localStorage 儲存。」

#### 打磚塊 Brick Breaker
- **初始 Prompt**：
  - 「使用 HTML Canvas 和原生 JavaScript 建一個打磚塊遊戲。包含球、板子、磚塊陣列。操作方式為左右方向鍵控制板子，球碰到磚塊後消失並得分，漏接球則扣一條生命，生命歸零時遊戲失敗。」
- **迭代 Prompt**：
  - 「在畫面右上角新增計分與剩餘生命顯示。」
  - 「加入多關卡設計：完成一層磚塊後增加球速並生成新磚塊。」
  - 「增加音效與粒子特效，打掉磚塊時播放音效與簡單動畫。」

#### 反應力測試 Reaction Timer
- **初始 Prompt**：
  - 「製作一個反應力測試網頁。畫面顯示一個按鈕，點擊『開始』後在 2~5 秒間隨機變色，變色後玩家需立刻按下，計算反應時間（毫秒）。顯示最近一次反應時間與歷史平均。」
- **迭代 Prompt**：
  - 「加入作弊防護：若玩家在變色前按下，顯示提醒並重新計時。」
  - 「新增排行榜區塊，顯示 5 次最佳成績，使用 localStorage 儲存。」
  - 「提供切換主題（亮色/暗色），並記住使用者的選擇。」

#### 猜數字 Number Guess
- **初始 Prompt**：
  - 「建立一個 1 到 100 猜數字的網頁遊戲。玩家輸入數字後顯示提示（太高或太低），記錄猜測次數，猜中後顯示恭喜訊息與總共花了幾次。」
- **迭代 Prompt**：
  - 「新增難度選擇：簡單（1-50）、普通（1-100）、困難（1-500），並根據難度決定可猜測次數上限。」
  - 「加入提示按鈕，每局可以使用一次，提示範圍縮小至剩餘範圍的一半。」
  - 「提供分享連結功能，完成後可以複製結果與難度資訊。」

#### 貪食蛇 Snake Game
- **初始 Prompt**：
  - 「請使用 HTML Canvas 與 JavaScript 建置貪食蛇遊戲。蛇可以用上下左右控制，吃到食物身體變長、得分增加，撞到牆或自己則遊戲結束，顯示最終得分。」
- **迭代 Prompt**：
  - 「加入速度調整：隨得分提高逐步增加移動速度。」
  - 「增加暫停功能，按下空白鍵可以暫停與恢復。」
  - 「新增排行榜與最佳成績紀錄，使用 localStorage 儲存並顯示前三名。」
