# Task 拆解模板

```markdown
# Task Plan

| Task | Owner | Status | 說明 |
| ---- | ----- | ------ | ---- |
| 初始化專案結構 | AI | 待執行 | 建立必要資料夾與主程式檔案 |
| 撰寫資料讀取模組 | AI | 待執行 | 建立處理輸入文件的函式與驗證 |
| 生成統計報表 | AI | 待執行 | 完成統計計算與資料格式 |
| 視覺化輸出 | AI | 待執行 | 產生圖表並輸出檔案 |
| 撰寫測試與 README | AI | 待執行 | 撰寫測試案例與使用說明 |
```

> 使用方式：在 Gemini CLI 中要求 AI 產出 `task.md` 時，可請它參考此模板格式。
