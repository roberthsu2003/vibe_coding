# PRD｜命令列待辦工具

## 目標
建立一個可在終端機執行的待辦事項管理程式，支援新增、列出、完成與刪除任務。

## 使用者故事
- 作為使用者，我想新增待辦事項，以便追蹤工作。
- 作為使用者，我想查看目前待辦列表，了解尚未完成的事項。
- 作為使用者，我想標記任務為完成或刪除已不需要的項目。

## 功能需求
1. **新增任務**：
   - 指令格式：`todo add "任務內容"`
   - 必須回傳任務 ID 與內容
2. **列出任務**：
   - 指令格式：`todo list`
   - 依照建立時間排序，顯示 `ID、狀態、內容`
3. **完成任務**：
   - 指令格式：`todo done <ID>`
   - 更新任務狀態並顯示完成訊息
4. **刪除任務**：
   - 指令格式：`todo delete <ID>`
   - 成功刪除後顯示確認訊息

## 非功能需求
- 程式需以 Python 實作，支援 3.10 以上版本
- 任務資料儲存在本地 JSON 檔案（預設 `data/todos.json`）
- 指令錯誤時需提供友善訊息

## 驗收標準
- 以 `pytest` 撰寫至少 3 項單元測試，涵蓋新增、完成、刪除流程
- 執行 `python main.py` 或 `uv run main.py` 可顯示使用指令說明
- README 包含安裝與使用說明
