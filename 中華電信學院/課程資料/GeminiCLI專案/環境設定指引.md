# 環境設定指引

## 1. 建立專案資料夾
```bash
mkdir -p gemini_cli_projects
cd gemini_cli_projects
```

## 2. 初始化 `uv`
```bash
uv init
uv sync
source .venv/bin/activate  # Windows 使用 .venv\Scripts\activate
```

## 3. 安裝必要套件
```bash
uv add google-genai pandas matplotlib rich typer pytest
```

## 4. 設定 Gemini 環境變數
```bash
export GEMINI_API_KEY="你的 API Key"  # Windows 使用 set 或 PowerShell $env:GEMINI_API_KEY
```

## 5. 建議的資料夾結構
```text
gemini_cli_projects/
├── data/
│   └── sales_sample.csv
├── reports/
├── src/
│   ├── __init__.py
│   ├── main.py
│   ├── data_loader.py
│   ├── analyzer.py
│   └── visualizer.py
├── tests/
│   └── test_analyzer.py
├── PRD.md
└── task.md
```

## 6. 常用指令
- `uv run python src/main.py --help`
- `genai chat --model gemini-1.5-pro --text "說明當前任務進度"`
- `pytest`

## 7. 疑難排解
- 若 `genai` 指令無法使用，確認虛擬環境是否已啟動
- `ModuleNotFoundError`：重新執行 `uv sync`
- API 呼叫失敗：檢查網路連線與 API Key 是否有效
