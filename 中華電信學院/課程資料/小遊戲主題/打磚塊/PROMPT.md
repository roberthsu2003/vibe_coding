# 打磚塊｜Prompt 指南

## 初始需求
「使用 HTML Canvas 和原生 JavaScript 建一個打磚塊遊戲。包含球、板子、磚塊陣列。操作方式為左右方向鍵控制板子，球碰到磚塊後消失並得分，漏接球則扣一條生命，生命歸零時遊戲失敗。」

## 迭代需求
1. 「在畫面右上角新增計分與剩餘生命顯示。」
2. 「加入多關卡設計：完成一層磚塊後增加球速並生成新磚塊。」
3. 「增加音效與粒子特效，打掉磚塊時播放音效與簡單動畫。」

## 測試建議
- 測試左右鍵與滑鼠控制是否順暢
- 確認過關後球速增加且產生新關卡
- 檢查生命歸零後是否跳出結束畫面

## 延伸挑戰
- 加入道具掉落（加速、放大板子等）
- 支援觸控操作
- 統計最高分並記錄於 localStorage
