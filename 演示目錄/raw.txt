<p><span style="vertical-align:baseline">The </span><code style="vertical-align:baseline">.toml</code><span style="vertical-align:baseline"> file provides a powerful and structured base on which to build extensive support for complex commands. To help support a wide range of users, we made the required keys minimal (just </span><code style="vertical-align:baseline">prompt</code><span style="vertical-align:baseline">). And we support easy-to-use args with </span><code style="vertical-align:baseline">{{args}}</code><span style="vertical-align:baseline"> and shell command execution </span><code style="vertical-align:baseline">!{...}</code><span style="vertical-align:baseline"> directly into the prompt.</span></p>

<p><span style="vertical-align:baseline">The name of a command is determined by its file path relative to the </span><code style="vertical-align:baseline">commands</code><span style="vertical-align:baseline"> directory. Sub-directories are used to create </span><em><span style="vertical-align:baseline">namespaced commands</span></em><span style="vertical-align:baseline">, with the path separator (</span><code style="vertical-align:baseline">/</code><span style="vertical-align:baseline"> or </span><code style="vertical-align:baseline">\</code><span style="vertical-align:baseline">) being converted to a colon (</span><code style="vertical-align:baseline">:</code><span style="vertical-align:baseline">).</span></p>
<ul>
<li style="list-style-type:disc;vertical-align:baseline">
<p><span style="vertical-align:baseline">A file at </span><code style="vertical-align:baseline">&lt;project&gt;/.gemini/commands/test.toml</code><span style="vertical-align:baseline"> becomes the command </span><code style="vertical-align:baseline">/test</code><span style="vertical-align:baseline">.</span></p>
</li>
<li style="list-style-type:disc;vertical-align:baseline">
<p><span style="vertical-align:baseline">A file at </span><code style="vertical-align:baseline">&lt;project&gt;/.gemini/commands/git/commit.toml</code><span style="vertical-align:baseline"> becomes the namespaced command </span><code style="vertical-align:baseline">/git:commit</code><span style="vertical-align:baseline">.</span></p>
</li>
